1014. 数列操作

给定一个长度为N的整数数列A，并且可以对这个数列进行一种操作：将某个区间里的数全部+1或-1。再给一个长度相同的整数数列B，请问最少需要几次可以将A变成B？

输入格式

输入共三行。

第一行一个整数N

第二行是数列A

第三行是数列B

输出格式

输出共一行，包含一个正整数，表示最少需要的次数。

样例输入

3  
3 4 5
6 7 8

样例输出

3

数据范围

对于100%的数据，$N \le 10^5，|A_i|, |B_i| <= 10^6$

时空磁盘限制（运行时）

时间限制： 1000 ms

内存空间限制： 256 MiB

磁盘空间限制： 无限制

解题思路

拿到这道题，第一个能想到的操作自然是先把两个数组相减，得到一个新的数组，记为c[N]。

那么我们的目标变转换为：如何将数组d[N]中的所有数经过最少次操作变为 0 。

但然后呢？我们来观察下这个操作：将某个区间里的数全部+1或-1

⚠️区间指的是数组中连续的部分，那这个操作目前看来很有局限性啊！怎么办捏？

我不认为我可以用天才以外的词形容第一个想出差分法的人。

设差分数组为d[n], 则

需要注意的是，差分数列的第一项是原数列的第一项（否则你和谁差分）

那么目标“如何将数组d[N]中的所有数经过最少次操作变为 0”最终就变成了:如何用最少次操作把这个差分数列变为0.

充分性：

  • $d[i]=0,\forall i\ge1$ 这意味着原数组d中所有数都相等，等于第一个数$d[0]$。
  • $d[0]=0$ 这意味着数组d中所有数都等于0。

必要性显然。

我们再来观察下这个操作的作用：将某个区间里的数全部+1或-1。（⚠️区间指的是数组中连续的部分）

考察原数组 $c[i]$ 到 $c[j-1]$

  • $1\le i<s-1<s<j-1 < N -1$, (为方便说明，N > 7)如果把这一段数全部 +1，则对应到差分数组d中，除了$d^{’}[i]=d[i]+1,d^{’}[j]=d[j]-1$，其他的数全部不变，因为

$$ \begin{equation} d^{’}[s]=c[s]+1 - (c[s-1]+1)=d[s] \end{equation} $$

​ 而下标在$[i，j]$范围之外的数组元素自然没有变化。

  • $j = N -1$，根据（1）的构造方式，这种情况下，仅有$d[i]$会加（或减1），其余数组元素均不变。

😊经过这种转变后，我们可以对数组中任意两个元素，或是任意两个元素进行操作，从而跳出了区间的约束！

从而问题就变得明朗起来，我们分两步走：

  1. 只要借助第一条性质，我们可以把差分数组中某一符号的数全部变为0。这取决于正数之和是否大于负数绝对值之和，若是则可把负数全部变为0，否则相反。
  2. 用性质二把剩下的数全部归 0。

那么，操作次数K满足： $$ \begin{equation} K=min(SOP，abs(SOM))+abs(SOP-abs(SOM))=max(SOP，abs(SOM)) \end{equation} $$

其中 $SOP,SOM$ 分别意味着差分数列 d 中的正数之和和负数之和。 DONE✌️

上代码：

#include<iostream>
using namespace std;
int main(){
    int n = 0;
    cin >> n;

    if(n == 0){
        cout << 0;
        return 0;
    }

    long positive = 0;
    long minus = 0;
    long temp = 0;

    long a[n],b[n],d[n]; //这里可以让n等于足够大的常量，规范但是影响性能

    for(int i = 0; i < n; ++i){
        cin >> a[i];
    }

    for(int i = 0; i < n; ++i){
        cin >> b[i];
    }

    for(int i = 0; i < n; ++i){
        d[i] = a[i] - b[i];
    }
		
  	// 获取正数之和 和 Abs(负数之和)
    for(int i = 1; i < n; ++i){
        temp = d[i] - d[i - 1];
        if(temp > 0){
            positive += temp;
        } else {
            minus += -(temp);
        }
    }
  	
  	// 对第一个元素特殊处理
    if(d[0] > 0){
        positive += d[0];
    } else {
        minus += -(d[0]);
    }

    cout << max(positive,minus) << endl;

}

1033. 行列式求值

现在给出一个n阶行列式，求出它的值。

输入格式

第一行一个数n表示行列式阶数 接下来是一个n行n列的由整数构成的行列式

输出格式

一行打印一个int表示行列式的值

样例输入

3  
5 1 2  
4 2 5  
3 4 1

样例输出

-59

数据范围

$1\leq n\leq 101≤n≤10$

答案在int范围之内

时空磁盘限制（运行时）

时间限制： 1000 ms

内存空间限制： 512 MiB

磁盘空间限制： 无限制

解题思路

这道题的出发点应该是行列式的计算方法（我能想到的）

  • 代数余子式
  • 逆序数排列
  • 变换矩阵求对角线元素乘积

第三个方法基本上可以先排除，因为涉及到分数，意味着要转换int为float，估计是会有精度损失的，肯定过不了。

此外，老师还要求不要用STL中的库，比如vector,list,map,set之类的, 那就是说如果要用类似结构的话还要自己写一个。

逆序数法

先看公式： $$ \begin{equation} Det=\left|\begin{array}{cccc} a_{11} & a_{12} & \cdots & a_{1 n} \ a_{21} & a_{22} & \cdots & a_{2 n} \ \vdots & \vdots & \ddots & \vdots \ a_{n 1} & a_{n 2} & \cdots & a_{n n} \end{array}\right|=\sum(-1)^{\tau\left(j_1 \cdots j_n\right)} a_{1 j_1} \cdots a_{n j_n} \end{equation} $$ 记 $r=\tau\left(j_1 \cdots j_n \right)$， 用数组 $J$ 储存 $j_1 \cdots j_n $

用逆序数法实现行列式求值，必须实现三个函数：

  • reverse(数组J的长度，数组J) 求该数组所代表的一种排列情况下的逆序数大小
  • cofactorCalc(待求行列式的矩阵，待求行列式矩阵的维度，数组J) 计算给定一组 $J$ 排列的情况下 $(-1)^{\tau\left(j_1 \cdots j_n\right)} a_{1 j_1} \cdots a_{n j_n}$ 的值
  • perm(数组a, 数组长度)
