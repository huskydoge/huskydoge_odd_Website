T285399 耍赖皮

题目背景

小明和小刚进行游戏比赛，游戏规则如下：

  1. 两人从一串数字nums中挑选两个数字。小刚必须挑选小明选择数字之后的数字，如果小明选择最后一个数字，那么小刚获胜。
  2. 小刚被允许“耍赖皮”，也就是说，小明的数字必须大于2倍小刚的数字，才算小明获胜。
  3. 总之，假如小明选定第i位数字，小刚选定第j位数字，满足$i < j $且$ nums[i] > 2*nums[j]$，那么认为小明获胜，反之小刚获胜。

题目描述

请计算小明获胜的种数。

输入格式

第一行是数组长度N，第二行N个数，表示数组中的元素。

输出格式

一行，小明获胜的种数。

输入输出样例

输入 #1

5
1 3 2 3 1

输出 #1

2

输入 #2

5
2 4 3 5 1

输出 #2

3

说明/提示

$1 <= nums.length <= 5 \cdot 10^5$

$-2^{8} <= nums[i] <= 2^{8} - 1$

Solution

法一：归并排序，在归并之前先数逆序对的个数

class Solution {
public: //这个函数做的是：1.数数组中逆序对数 2.对数组进行排序
    int reversePairsRecursive(vector<int>& nums, int left, int right) {
        if (left == right) {
            return 0;
        } else {
            int mid = (left + right) / 2;
            int n1 = reversePairsRecursive(nums, left, mid);
            int n2 = reversePairsRecursive(nums, mid + 1, right);
            int ret = n1 + n2;

            // 首先统计下标对的数量
            int i = left;
            int j = mid + 1;
            while (i <= mid) {
                while (j <= right && (long long)nums[i] > 2 * (long long)nums[j]) j++;
                ret += (j - mid - 1); // 注意，这里因为左右两边数组已经有序了
                i++;
            }

            // 随后合并两个排序数组
            vector<int> sorted(right - left + 1);
            int p1 = left, p2 = mid + 1;
            int p = 0;
            while (p1 <= mid || p2 <= right) {
                if (p1 > mid) {
                    sorted[p++] = nums[p2++];
                } else if (p2 > right) {
                    sorted[p++] = nums[p1++];
                } else {
                    if (nums[p1] < nums[p2]) {
                        sorted[p++] = nums[p1++];
                    } else {
                        sorted[p++] = nums[p2++];
                    }
                }
            }
            for (int i = 0; i < sorted.size(); i++) {
                nums[left + i] = sorted[i];
            }
            return ret;
        }
    }

    int reversePairs(vector<int>& nums) {
        if (nums.size() == 0) return 0;
        return reversePairsRecursive(nums, 0, nums.size() - 1);
    }
};

T285706 快乐起来！

题目背景

小明终于可以线下上课了，为了庆祝线下上课，他打算去做一些“快乐”的事。但是由于精力有限，他最多只能做k件事。最初他有一定的“快乐资本”，对于每件事，都有一个基础的“快乐底线”。当他的“快乐资本”值大于这件事的“快乐底线”时，他就可以选择做这件事，收获这件事得到的“快乐收益”值，将“快乐收益”加入到他的“快乐资本”中。

题目描述

给你n件事，每件事情都有一个“快乐收益”和“快乐底线”，以数组方式给出。还有一个值k，表示最多只能选择k件事；以及一个值w，代表最初的“快乐资本”。请你计算小明最多可以获得的“快乐资本”值。答案保证在 32 位有符号整数范围内。

输入格式

第一行是三个数字，分别代表k、w和n值。

第二行n个数字，代表“快乐收益”。

第三行n个数字，代表“快乐底线”。

输出格式

一个数字，代表最多可以获得的“快乐收益”。

输入输出样例

输入 #1

2 0 3
1 2 3
0 1 1

输出 #1

4
由于你的初始“快乐资本”为 0，你仅可以从 0 号项目开始。

在完成后，你将获得 1 的“快乐收益”，你的“快乐资本”将变为 1。

此时你可以选择开始 1 号或 2 号项目。

由于你最多可以选择两个项目，所以你需要完成 2 号项目以获得最大的资本。

因此，输出最后最大化的资本，为 0 + 1 + 3 = 4

输入 #2

3 0 3
1 2 3
0 1 2

输出 #2

6

说明/提示

$1 <= k <= 10^5$

$0 <= w <= 10^9$

$1 <= n <= 10^5$

$0 <= profits[i] <= 10^4$

$0 <= capital[i] <= 10^9$

Analysis

  • Struct Node 存储int profit, int cost，保存在数组node[]中

  • 将数组node按cost从小到大重排(sort)

  • 重排后将node依次入堆，规则如下：

    • 按profit大小建堆（大根堆）

    • 如果node[i].cost <= w, node[i+1].cost > w , 表示不能再入堆，此时停止入堆操作，开始弹出堆顶元素（w += heap.top().profit）直至w符合条件（即node[i+1].cost <= w）

      注意我们按cost从小到大入堆，保证了已入堆的元素一定可选，由于维护的是大根堆，直接弹出堆顶元素即可。

关于优先队列

优先队列是一种功能强大的队列。如果不太清楚队列，可以看看这篇博客 。

它的功能强大在哪里呢？ 四个字：自动排序。

优先队列的头文件&&声明

首先，你需要

#include<queue>
using namespace std;

其次，一个优先队列声明的基本格式是： priority_queue<结构类型> 队列名; 比如：

priority_queue <int> i;
priority_queue <double> d;

不过，我们最为常用的是这几种：

priority_queue <node> q;
//node是一个结构体
//结构体里重载了‘<’小于符号
priority_queue <int,vector<int>,greater<int> > q;
//不需要#include<vector>头文件

//注意后面两个“>”不要写在一起，“>>”是右移运算符
priority_queue <int,vector<int>,less<int> >q;

定义：priority_queue<Type, Container, Functional>

Type 就是数据类型，Container 就是容器 类型（Container必须是用数组实现的容器，比如vector,deque等等，但不能用 list。STL里面默认用的是vector），Functional 就是比较的方式。

Solution

#include <cstdio>
#include <algorithm>
#include <queue>
#include <vector>
using namespace std;

const int N = 1e5 + 10;

struct Node{
    int p;
    int c;
}node[N]; // 声明 + 构造

bool cmp(Node a, Node b){ //cmp用于sort函数
    return a.c < b.c;
}

bool operator<(const Node &a, const Node &b){ //重载运算符用于建堆
    return a.p < b.p;
}

int main(){
    int k, w, n;
    //data input
    scanf("%d%d%d", &k, &w, &n);
    for(int i = 0; i < n; ++i)
        scanf("%d", &node[i].p);
    for(int i = 0; i < n; ++i)
        scanf("%d", &node[i].c);
    sort(node, node+n, cmp); //按cost从小到大排序

    // for(int i = 0; i < n; ++i){
    //     printf("%d", node[i].c); 
    // }
    int num = 0;
    int flag = 0;
    priority_queue<Node, vector<Node>, less<Node> > q; //less表示大根堆(元素间比较方式为<, "<"需要重载)
    for(int i = 0; i < n ; ++i) {
        q.push(node[i]);
        while(node[i].c <= w && i < n-1 && node[i+1].c > w) { //考虑了i=1,...,n-2,所以下面i=n-1要单独讨论
            if(q.empty()) {
                flag = 1; // 直接结束, 表示不能选择完 k 件事
                break;
            }
            w += q.top().p;
            q.pop();
            if(++num == k) break;
        }
        if(num == k || flag) break;
        if(i == n-1 && node[n-1].c <= w) {
            w += q.top().p;
            // ++num;
            // q.pop();
        }
    }
    printf("%d", w);
    return 0;
}
